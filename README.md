# Breast-Cancer-Prediction

Description:-
Machine learning project applying logistic regression to classify breast cancer outcomes using structured medical datasets.

Tools:- Python, Pandas, NumPy, Scikit-learn

Work Performed:-

-Preprocessed and cleaned medical data

-Built a logistic regression classification model

-Evaluated performance using accuracy, recall, and F1-score

-Outcome: Demonstrated applied statistical modeling for healthcare analytics.
